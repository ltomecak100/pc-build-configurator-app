# PC Build Configurator (ConnectIT)

Software engineering project developed as part of the *Software Engineering* course at the Faculty of Organization and Informatics.

The goal of the project is to design a system that helps customers in computer hardware stores select compatible PC components and build computer configurations.

The system prevents incompatible component combinations and improves the purchasing experience in IT retail environments.

---

## Project Overview

Customers often struggle to choose compatible computer components when building a PC.  
The ConnectIT application solves this problem by providing a system that:

- displays a catalog of available components
- automatically checks compatibility between components
- allows users to create custom PC configurations
- generates reports for selected configurations
- synchronizes product data with the store database

The application is designed to run on kiosk computers inside the store.

---

## Features

- Authentication for employees
- Product catalog management
- Compatibility validation between components
- PC configuration builder
- Configuration report generation
- Email export of configuration
- Synchronization with existing store database
