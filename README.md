# pc-build-configurator-app
Software engineering project for selecting compatible PC components and building computer configurations, including requirements analysis, UML diagrams, and C# implementation.

## Project Summary

ConnectIT is a software system designed for computer hardware stores.

The application helps customers build compatible PC configurations by selecting components from a product catalog.

The system automatically validates component compatibility and generates configuration reports.

The project includes requirements analysis, UML diagrams, and system design documentation.

## Note

The full project documentation was originally written in Croatian as part of a university course project.
Key diagrams and system architecture are included in this repository.
